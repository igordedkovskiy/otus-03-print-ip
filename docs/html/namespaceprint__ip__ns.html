template<print_ip_helpers::Integer T>
void print_ip_ns::print_ip ( const T &  addr)

Prints addr like it's an ip address.

If T is an integer, prints addr's bytes in reverse order, e.g. starting from higher byte, separated by dots.

If T is std::vector<ip_field_t> or std::list<ip_field_t>, where t is an integer, prints elements of T one by one, separated by dots.
